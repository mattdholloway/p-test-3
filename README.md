## Getting Started

### Prerequisites

- Git installed on your local machine
- Access to this GitHub repository

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mattdholloway/p-test-3.git
   cd p-test-3
   ```

2. Switch to the development branch:
   ```bash
   git checkout develop
   ```
